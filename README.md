# Playwright E2E Automation Framework

## Overview
This project is an **End-to-End automation testing framework built using Playwright with JavaScript**.  
It automates a complete user journey such as **login, product selection, checkout, and order confirmation**, following basic automation best practices.

This project is intended to showcase **practical Playwright automation skills** and a **clean Page Object Model structure**.

---

## Tech Stack
- Playwright
- JavaScript (Node.js)
- Playwright Test Runner
- Page Object Model (POM)
- Git & GitHub

---

## Key Features
- End-to-End test coverage for a real user flow
- Page Object Model for better maintainability
- External test data support (JSON)
- Stable assertions and waits
- Clean and easy-to-understand project structure
